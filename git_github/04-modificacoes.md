# ver modificações nos arquivos

O git diff mostra as diferenças entre versões do código no Git.(antes de dar commit)             
``` bash
git diff
```

Ver o que foi adicionado ao staged, o que está pronto para o commit:
```bash
git diff --staged
```

compara dois commits:
```bash
git diff commit1 commit2
```
Comparar branches:
```bash
git diff main feature
```
